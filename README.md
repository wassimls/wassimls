### &nbsp;`about me`

&nbsp;&nbsp;I'm a **UI/UX Designer** and **Frontend Developer** who builds\
&nbsp;&nbsp;premium digital experiences from the ground up — no templates,\
&nbsp;&nbsp;no shortcuts. Every pixel is intentional.

&nbsp;&nbsp;I work at the intersection of **design systems**, **motion**,\
&nbsp;&nbsp;and **creative frontend engineering** — using tools like\
&nbsp;&nbsp;Three.js and GSAP to push what a landing page can feel like.

&nbsp;&nbsp;**What drives me:**
- Building luxury, editorial-grade interfaces
- Motion design that guides without distracting  
- Typography and whitespace as design tools
- Turning Figma files into living, breathing code
